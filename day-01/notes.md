# Day 01 - Terminal & Setup

## What I learned today

### Terminal Commands
- pwd: shows where I am
- ls: lists files, ls -la shows hidden files and details
- cd: moves between folders, cd ~ goes home
- mkdir: creates folders, -p creates parent folders too
- touch: creates empty files
- echo "text" > file: writes text into a file
- cat: reads file contents
- cp: copies files
- mv: moves or renames files
- rm: deletes files permanently (no recycle bin!)
- clear: cleans the screen
- history: shows all commands I've typed

### Tools Installed
- Homebrew: package manager for Mac (installs other tools)
- Node.js: runs JavaScript outside the browser (needed for web apps)
- Git: tracks changes to my code (like a save system with history)
- Cursor: AI-powered code editor (where I'll write code with AI help)
